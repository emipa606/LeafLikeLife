# Smokeleaf changes

**Note**: *The following changes can be customized in the mod settings*



-  Consciousness penalty reduced from 30% to 15%
-  Customize the movement, sleepiness, and hunger penalties
-  Half as addictive
-  Asthma chance decreased by half
-  Carcinoma chance decreased by three-quarters
-  Withdrawal mood penalty reduced from -22 to -8
